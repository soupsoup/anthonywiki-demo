**Gamma** is an AI-powered design tool that generates presentations, websites, documents, and social media graphics from a text prompt. The pitch is end-to-end AI generation — not AI-assisted templates, but a full first draft produced from a description — combined with multi-format output from a single platform.

The API is the most strategically interesting feature for content and newsletter operations: you can pipe data into Gamma and get polished decks or pages out programmatically, which opens up automated reporting and visual summary pipelines.

## What it generates

**Presentations** — AI-generated slides with consistent branding throughout. Exportable to PowerPoint and Google Slides. Useful for recaps, briefings, and visual summaries that would otherwise require manual slide-building.

**Websites** — landing pages, portfolios, and business sites with no coding required. Generated from a description, editable in a visual interface.

**Documents** — proposals, PDFs, and visual reports. The AI handles layout and hierarchy, not just copy.

**Social media** — copy and graphics sized for platforms, shareable directly without exporting.

## API

Gamma exposes a programmatic API for automating content creation at scale. This is the most compelling angle for anyone running content operations: instead of manually generating decks, you can build a pipeline where structured data flows in and polished Gamma output flows out automatically. Use cases include:

- Weekly performance summaries generated from analytics exports
- Newsletter visual editions built from editorial data
- Automated investor or stakeholder update decks

The API positions Gamma as content infrastructure rather than just a consumer tool.

## Competitive landscape

| Tool | Strength | Gap |
|------|----------|-----|
| Gamma | Speed + multi-format + API | Less design control |
| Canva | Brand templates + asset library | Weaker AI generation |
| Beautiful.ai | Smart slide layouts | No websites or docs |
| Tome | Narrative flow | Smaller output surface |
| PowerPoint/Google Slides | Ubiquitous, full control | Manual, no AI generation |

Gamma's differentiation is the combination of deep AI generation (not just AI-assisted) with breadth of output format. The API is a meaningful moat that consumer-focused tools like Canva haven't fully matched.

## Business model

Freemium — free tier for individual use, paid tiers for teams and heavier usage.
